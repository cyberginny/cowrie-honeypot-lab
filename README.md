# Cowrie Honeypot Lab (Cybersecurity Project)

This project demonstrates the deployment, attack simulation, and log analysis
of a Cowrie SSH honeypot in a controlled lab environment.

## Tools Used
- Cowrie Honeypot
- Ubuntu / Debian
- Kali Linux
- Nmap
- Hydra
- Wireshark
- tcpdump

## Project Structure
- `honeycowrie_installation.sh` – Setup and attack simulation commands
- `cowrie logs` – Authentication and attack logs

## Lab Steps
1. Environment setup
2. Network reconnaissance
3. Simulated SSH attacks
4. Traffic capture
5. Log analysis

## Disclaimer
This project is for **educational and ethical cybersecurity research only*
